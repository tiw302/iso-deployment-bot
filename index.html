> project_summary
[overview]
automated iso mirroring system designed to synchronize multiple operating system
distributions to google drive storage. the system utilizes github actions for
scheduled execution, aria2 for multi-threaded downloads, and rclone for cloud
storage synchronization.

[why this exists]
when deploying operating systems across virtualized environments (like proxmox,
hyper-v) or physical machines, fetching iso images from slow, geographically
distant upstream mirrors can waste hours of time.

this library maintains high-speed, direct google drive mirrors of the most
common, secure, and clean system installation media. it guarantees that all
deployment files are:
 1. hosted on enterprise-grade cloud storage for maximum download bandwidth.
 2. verified and synced daily to prevent broken links or stale versions.
 3. stored securely, providing a single consolidated inventory.
