[ ข้อมูลโครงการคลังทรัพยากรซอฟต์แวร์เสรี - os-deployment-library ]

สวัสดีครับคุณครู หรือพี่ๆ แอดมินระบบทุกท่านที่มาอ่านนะครับ

โฟลเดอร์นี้คือโปรเจกต์ส่วนตัวที่ผมจัดทำขึ้นเพื่อการศึกษาเกี่ยวกับระบบปฏิบัติการ Open Source (Linux Distros) และการจัดการระบบจัดเก็บข้อมูลอัตโนมัติ (Automation Storage) โดยใช้ Google Colab และ Gitub Action (main) คับด้วยภาษา python

[ วัตถุประสงค์ ]
    เพื่อรวบรวมไฟล์ติดตั้ง OS ที่เป็น Open Source 100% (ถูกกฎหมาย) ไว้ใช้ศึกษาในโรงเรียน
    เพื่อฝึกฝนการเขียน Script และการใช้เครื่องมือจำพวก aria2 / rclone ในระดับสูง
    เพื่อศึกษาพฤติกรรมการรับส่งข้อมูล (Traffic) ของเซิร์ฟเวอร์กระจายไฟล์
